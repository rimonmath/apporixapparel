# khudroshop

This template is a combined project of vue + hono.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Deployment

Only dist folder, node_modules, .env , hono.min.js and package.json is needed after build.

```sh
node hono.min.js
```
